## 📂 Sobre o Repositório

Este repositório reúne **queries SQL** desenvolvidas para o **Microsoft SQL Server**. Os scripts foram criados para fins de estudo, prática e referência pessoal, abordando diferentes operações e funcionalidades do SQL Server.

### 📋 Tipos de Queries

| Categoria | Descrição |
|-----------|-----------|
| **DDL** *(Data Definition Language)* | `CREATE`, `ALTER`, `DROP` — criação e alteração de tabelas, índices e estruturas |
| **DML** *(Data Manipulation Language)* | `INSERT`, `UPDATE`, `DELETE` — manipulação e modificação de dados |
| **DQL** *(Data Query Language)* | `SELECT` com filtros, junções (`JOIN`), subconsultas e agregações |
| **DCL** *(Data Control Language)* | `GRANT`, `REVOKE` — controle de permissões e acessos |
| **TCL** *(Transaction Control Language)* | `BEGIN TRANSACTION`, `COMMIT`, `ROLLBACK` — controle de transações |
| **Stored Procedures** | Procedimentos armazenados para lógica de negócio no banco |
| **Funções** | Funções escalares e de tabela definidas pelo usuário |
| **Views** | Visões para simplificação de consultas complexas |
| **Triggers** | Gatilhos para automação de ações no banco de dados |

### 🎯 Objetivo

O objetivo deste repositório é documentar e compartilhar scripts SQL úteis do dia a dia, que possam servir como referência para:

- 📌 Consultas e relatórios
- 🏗️ Modelagem e criação de bancos de dados
- ⚙️ Otimização de performance
- 🔐 Controle de acesso e segurança
- 🔄 Automação com procedures e triggers

## 🚀 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Wssfilho/DB-querys.git
   ```

2. Abra o arquivo `.sql` desejado no **SQL Server Management Studio (SSMS)** ou em qualquer ferramenta compatível com T-SQL.

3. Execute a query no banco de dados desejado.

> ⚠️ **Atenção:** Sempre revise os scripts antes de executá-los em ambientes de produção.
